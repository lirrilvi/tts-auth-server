========================================
  ИНСТРУКЦИЯ ДЛЯ ЗАПУСКА НА WINDOWS
========================================

ЧТО ЭТО: сервер токенов + Telegram бот для TTS расширения.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ШАГ 1 — УСТАНОВИТЬ PYTHON
─────────────────────────
1. Зайди на https://python.org/downloads
2. Скачай Python 3.12 или новее
3. При установке ОБЯЗАТЕЛЬНО поставь галочку:
   ☑ Add Python to PATH
4. Нажми Install Now

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ШАГ 2 — ЗАПОЛНИТЬ ФАЙЛ .env
────────────────────────────
Открой файл .env в этой папке (Блокнот или любой редактор).
Он уже заполнен — ничего менять не нужно.

Если файл не видно — включи показ скрытых файлов:
Проводник → Вид → Скрытые элементы ☑

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ШАГ 3 — ЗАПУСТИТЬ
──────────────────
Запускать нужно ДВА файла (каждый в своём окне):

  start_server.bat  ← запускает сервер токенов
  start_bot.bat     ← запускает Telegram бота

При первом запуске установятся все нужные библиотеки
(интернет нужен только при первом запуске).

В окне сервера должно появиться:
  ✅ Сервер запущен: http://localhost:5055

В окне бота:
  🤖 Бот запущен...

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ШАГ 4 — ДЕРЖАТЬ ЗАПУЩЕННЫМ
────────────────────────────
Оба окна должны быть открыты пока бот работает.
Можно свернуть — закрывать нельзя.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
────────────────────────
• Ошибка "python не найден" → переустанови Python с галочкой Add to PATH
• Порт занят → перезагрузи компьютер и попробуй снова
• Бот не отвечает → проверь что оба окна открыты
